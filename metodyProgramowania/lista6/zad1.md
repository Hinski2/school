# zad1

* trzeba zrobić dune build w folderze calc
* w terminalu klikamy: dune utop
* w utopie kilkamy: open Calc.Main;;
* możemy teraz dać dowole przykłady
``` ocaml
    interp "if 2 = 3 then 1 else 10 + 100";;
    interp " 1 = 3    ";;
    interp " 2 + 2 * 2";;
    interp " 3 / 100 * 100";;
    interp "    if cos then cos else 3 / 100    ";;
```
